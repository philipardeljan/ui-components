What is UI Components?

UI components is a 4kb functional CSS file. Sitting on top of that CSS file are hundreds of UI components.

Why does it exist?

There is too much CSS out there. 15kb of CSS is all you'll ever need. If your minified and compressed CSS file is over 100kb, you are doing something wrong. At least, that's the view I have and wish others did too.

What's new about this?

Functional CSS moves the burden of styling your application from CSS to HTML. This at first may seem like a terrible idea (see separation of concerns) but it really isn't. The most DRY code you can hope to get is to combine functional CSS with a templated front end view (.net/rails partials, react components etc.)

Everything in the end compiles back to raw HTML and CSS.
